# AI Legal Sentinel

AI Legal Sentinel is a privacy-first AI-powered system designed to help freelancers
and startups analyze contracts under Indian law. It detects risky clauses,
explains them in simple language, and provides a clear risk score.

## Problem Statement
Freelancers often sign contracts containing complex legal clauses they do not
fully understand. Legal consultation is expensive and inaccessible to many.

## Features
- Contract upload (PDF/DOCX)
- Clause-level analysis
- Indian law-based risk detection
- Explainable AI output
- Privacy-first design (no data storage)

## Tech Stack
- Backend: FastAPI (Python)
- Frontend: React
- AI/NLP: Rule-based analysis + LLM explanations

## Disclaimer
This system provides informational analysis only and does not constitute legal advice.
